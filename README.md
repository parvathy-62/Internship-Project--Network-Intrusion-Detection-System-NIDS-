# NARL-network-intrusion-detection-system
The rapid growth of digital communication and internet-based services has significantly increased the volume of network traffic generated worldwide. While technological advancements have improved connectivity and accessibility, they have also introduced new cybersecurity challenges. Modern computer networks are continuously exposed to threats such as Denial-of-service attacks, unauthorized access attempts, probing activities, and various forms of malicious intrusions.
Network Intrusion Detection Systems (NIDS) serve as an essential layer of defence by monitoring network traffic and identifying suspicious activities. Traditional intrusion detection systems primarily rely on signature-based detection techniques, which compare network events against predefined attack signatures. Although effective for known threats, these systems often struggle to identify previously unseen attacks and evolving threat patterns.
Machine Learning (ML) has emerged as a promising solution for intrusion detection by enabling systems to learn patterns directly from historical network data. By analysing network traffic characteristics, ML algorithms can automatically classify connections as normal or malicious and adapt to new attack behaviours without relying solely on manually crafted signatures.
This project explores the application of ML techniques for developing a NIDS using the NSL-KDD benchmark (supervised)dataset. The study investigates both binary intrusion detection and multiclass attack classification, with the objective of building accurate and reliable models for automated network security monitoring.
